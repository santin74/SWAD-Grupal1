Estudio del dataset plasma retinol y Betacaroteno
=================================================
Trabajo Grupal para Software y análisis de datos
------------------------------------------------

Se ha seleccionado la base de datos “Plasma Retinol and Beta-Carotene Dataset” seleccionada de la Universidad de Vandervilt,
Nashvile, Tenesee (<http://biostat.mc.vanderbilt.edu/wiki/Main/DataSets>)
* En estos datos se recoge información cualitativa de los participantes en el estudio,
así como parámetros de estudio relacionadas con su nutrición y uso de vitaminas.

* Especialmente interesante para el estudio es la registro de niveles de
betacaroteno y retinol ingerido y asimilado en sangre por los participantes.
